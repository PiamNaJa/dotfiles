# Epoch Conversion Changelog

## [Initial Version] - 2025-11-25

- Initial release of Epoch Converter
- Convert between Timestamp and Date formats
- View current time in multiple formats (Local, UTC, ISO 8601)
- Support for both Seconds and Milliseconds timestamps
- Improved timestamp detection (auto-detect seconds vs milliseconds)
- Updated local date format to YYYY-MM-DD HH:MM:SS
